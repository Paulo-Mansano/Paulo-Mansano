## Projetos em Destaque

### Plataforma Web de Gerenciamento de RPG (Pokémon)
Um sistema robusto para gestão de campanhas de RPG focado no universo Pokémon.
* **O Desafio:** Lidar com uma lógica de negócios extensa e um volume alto de relações dinâmicas (status, inventário, cálculos em tempo real) sem comprometer a performance.
* **A Solução:** Desenvolvi o back-end em **Java** com uma modelagem relacional altamente estruturada. Realizei uma migração arquitetural para o **Supabase**, utilizando o **PostgreSQL** de forma avançada. O grande diferencial foi a reescrita e otimização pesada de *queries* SQL para reduzir a latência e contornar as restrições computacionais de servidores gratuitos.

### Pesquisa em IA Aplicada a Dados Biométricos
Projeto de Iniciação Científica (PUCRS) focado em extrair inteligência de dispositivos vestíveis.
* **O Desafio:** Lidar com dados "sujos" e não-estruturados capturados por sensores no mundo real.
* **A Solução:** Estruturação dos dados e aplicação de modelos preditivos e de classificação (*KNN, Random Forest, Naïve Bayes*) para identificar padrões de risco à saúde, como quedas e eventos cardíacos.
